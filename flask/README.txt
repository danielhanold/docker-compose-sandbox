This Docker environment uses the Flask development server and is based on this tutorial:
http://flask.pocoo.org/docs/0.12/tutorial/

The full source cod eis hosted on https://github.com/pallets/flask/tree/master/examples/flaskr.

To initialize the database, connect to the web container and execute this command: flask initdb
@see http://flask.pocoo.org/docs/0.12/tutorial/dbinit/#tutorial-dbinit
